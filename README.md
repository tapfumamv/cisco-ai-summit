# Cisco AI Summit 2026 - Decoded

An independent companion guide to the [Cisco AI Summit 2026 livestream](https://www.youtube.com/live/YO2PVbtpb_A), featuring AI-assisted analysis of every speaker session.

**Live site:** [tapfumamv.github.io/cisco-ai-summit](https://tapfumamv.github.io/cisco-ai-summit/)

## What is this?

A magazine-style breakdown of the Cisco AI Summit 2026 (San Francisco, February 3, 2026). Each speaker gets a narrative analysis and a quick-take summary, built from the public livestream transcript.

15 sessions covered - from Sam Altman and Marc Andreessen to the geopolitics panel and the builders from Anthropic, OpenAI, and Google.

## Features

- Narrative + Quick Take tabs per speaker
- Dark/light mode
- Responsive design
- Chronological and theme-based navigation
- Direct video links to each speaker's segment

## Disclaimer

This is **not affiliated with Cisco**. All analysis is based on the publicly available livestream. Speaker headshots are sourced from public corporate pages, press kits, and LinkedIn profiles and remain the property of their respective owners.

## Built with

Static HTML, CSS, and vanilla JavaScript. AI-assisted analysis and site generation. Hosted on GitHub Pages.
